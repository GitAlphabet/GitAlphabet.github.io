# Vuepress部署

# 部署到服务器

.vuepress下面的 config.js 里面配置
    base:'/dist/'

# 部署

a) 切换到 .vuepress文件夹下的 dist 目录
b) 执行下列代码(以放在 github 上的 username.github.io 仓库上为例)
    git init
    git add -A
    git commit -m'deploy'
    git push -f git@github.com:GitAlphabet/GitAlphabet.github.io.git master

# 脚本部署

创建 deploy.sh 脚本并添加以下命令

git init
git add -A
git commit -m'deploy'
git push -f git@github.com:GitAlphabet/GitAlphabet.github.io.git master

执行脚本

sh deploy.sh
