冒泡排序 ​

冒泡排序（Bubble Sort）是一种简单直观的排序算法。它的名字由来是因为排序过程中较大的元素会像气泡一样逐渐"浮"到数组的末尾。

算法思想 ​

  • 比较相邻的元素。如果第一个比第二个大，就交换他们两个。
  • 对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对。这步做完后，最后的元素会是最大的数。
  • 针对所有的元素重复以上的步骤，除了最后一个。
  • 持续每次对越来越少的元素重复上面的步骤，直到没有任何一对数字需要比较。

基础实现 ​

javascript
/**
 * 基础冒泡排序
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function bubbleSort(arr) {
    let len = arr.length;
    // 外层循环控制排序轮数
    for (let i = 0; i < len - 1; i++) {
        // 内层循环进行相邻元素比较和交换
        // 每轮结束后，最大的元素会冒泡到最后，所以内循环可以减少 i 次
        for (let j = 0; j < len - 1 - i; j++) {
            // 如果前一个元素大于后一个元素，则交换
            if (arr[j] > arr[j + 1]) {
                // 使用解构赋值进行元素交换
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
            }
        }
    }
    return arr;
}

const array = [5, 3, 7, 2, 9, 1, 6];

优化版冒泡排序 ​

javascript
/**
 * 优化版冒泡排序
 * 如果某一轮没有发生交换，说明数组已经有序，可以提前结束
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function optimizedBubbleSort(arr) {
    let len = arr.length;
    // 添加一个标志位，判断是否发生了交换
    let swapped;
    
    for (let i = 0; i < len - 1; i++) {
        swapped = false;
        for (let j = 0; j < len - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                [arr[j], arr[j + 1]] = [arr[j + 1], arr[j]];
                swapped = true;
            }
        }
        // 如果这一轮没有发生交换，说明数组已经有序
        if (!swapped) break;
    }
    return arr;
}

双向冒泡排序（鸡尾酒排序） ​

javascript
/**
 * 双向冒泡排序（鸡尾酒排序）
 * 从两端同时进行冒泡，可以加快排序速度
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function cocktailSort(arr) {
    let len = arr.length;
    let start = 0;
    let end = len - 1;
    
    while (start < end) {
        // 从左到右冒泡，最大的元素到右端
        for (let i = start; i < end; i++) {
            if (arr[i] > arr[i + 1]) {
                [arr[i], arr[i + 1]] = [arr[i + 1], arr[i]];
            }
        }
        end--;
        
        // 从右到左冒泡，最小的元素到左端
        for (let i = end; i > start; i--) {
            if (arr[i] < arr[i - 1]) {
                [arr[i], arr[i - 1]] = [arr[i - 1], arr[i]];
            }
        }
        start++;
    }
    return arr;
}

选择排序 ​

选择排序（Selection Sort）是一种简单直观的排序算法。它的核心思想是每次从待排序的数据中选出最小（或最大）的元素，存放在序列的起始位置。

算法思想 ​

  • 初始状态： 将序列中的第一个元素视为最小值。
  • 遍历未排序部分： 遍历剩余的未排序部分，找到其中最小的元素。
  • 交换操作： 将最小元素与未排序部分的第一个元素交换位置，即将其放到已排序部分的末尾。
  • 重复操作： 重复以上步骤，直到整个序列排序完成。

基础实现 ​

javascript
/**
 * 选择排序
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function selectionSort(arr) {
    const n = arr.length;
    // 外层循环控制排序轮数，每轮确定一个最小元素的位置
    for (let i = 0; i < n - 1; i++) {
        let minIndex = i; // 假设当前第一个元素最小
        // 内层循环在未排序部分寻找真正的最小元素
        for (let j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j; // 找到更小的元素，更新最小值索引
            }
        }
        // 将找到的最小元素与当前位置交换
        if (minIndex !== i) {
            [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
        }
    }
    return arr;
}

const array = [5, 3, 7, 2, 9, 1, 6];

双向选择排序 ​

javascript
/**
 * 双向选择排序
 * 同时选择最小和最大元素，减少排序轮数
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function doubleSelectionSort(arr) {
    const n = arr.length;
    let left = 0;
    let right = n - 1;
    
    while (left < right) {
        let minIndex = left;
        let maxIndex = left;
        
        // 在未排序部分寻找最小和最大元素
        for (let i = left + 1; i <= right; i++) {
            if (arr[i] < arr[minIndex]) {
                minIndex = i;
            }
            if (arr[i] > arr[maxIndex]) {
                maxIndex = i;
            }
        }
        
        // 将最小元素放到左端
        if (minIndex !== left) {
            [arr[left], arr[minIndex]] = [arr[minIndex], arr[left]];
        }
        
        // 如果最大元素原本在左端，已经被交换，需要调整maxIndex
        if (maxIndex === left) {
            maxIndex = minIndex;
        }
        
        // 将最大元素放到右端
        if (maxIndex !== right) {
            [arr[right], arr[maxIndex]] = [arr[maxIndex], arr[right]];
        }
        
        left++;
        right--;
    }
    return arr;
}

    • 平均情况：O(n²)

插入排序 ​

插入排序（Insertion Sort）是一种简单直观的排序算法。它的工作原理是通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。

算法思想 ​

  • 初始状态： 将序列中的第一个元素视为已排序部分。
  • 遍历未排序部分： 从未排序部分选择一个元素，将其插入到已排序部分的合适位置，使得已排序部分仍然有序。
  • 插入操作： 将选定的元素与已排序部分的元素逐个比较，找到合适的插入位置，并将其插入。
  • 重复操作： 重复以上步骤，直到整个序列排序完成。

基础实现 ​

javascript
/**
 * 插入排序
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function insertionSort(arr) {
    const n = arr.length;
    // 从第二个元素开始，逐个插入到已排序部分
    for (let i = 1; i < n; i++) {
        let current = arr[i]; // 当前待插入的元素
        let j = i - 1; // 已排序部分的最后一个元素的下标
        
        // 将当前元素与已排序部分比较，找到合适的位置
        // while循环条件：j >= 0 且当前元素小于已排序部分的元素
        while (j >= 0 && arr[j] > current) {
            // 将大于当前元素的元素向后移动
            arr[j + 1] = arr[j];
            j--;
        }
        // 将当前元素插入到正确位置
        arr[j + 1] = current;
    }
    return arr;
}

const array = [5, 3, 7, 2, 9, 1, 6];

优化版插入排序（二分插入） ​

javascript
/**
 * 二分插入排序
 * 使用二分查找确定插入位置，减少比较次数
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function binaryInsertionSort(arr) {
    const n = arr.length;
    
    for (let i = 1; i < n; i++) {
        let current = arr[i];
        let left = 0;
        let right = i - 1;
        
        // 使用二分查找确定插入位置
        while (left <= right) {
            const mid = Math.floor((left + right) / 2);
            if (current < arr[mid]) {
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        
        // 将插入位置后的元素向后移动
        for (let j = i - 1; j >= left; j--) {
            arr[j + 1] = arr[j];
        }
        
        // 插入元素
        arr[left] = current;
    }
    return arr;
}

链表插入排序 ​

javascript
/**
 * 链表节点类
 */
class ListNode {
    constructor(val, next = null) {
        this.val = val;
        this.next = next;
    }
}

/**
 * 链表插入排序
 * @param {ListNode} head - 链表头节点
 * @returns {ListNode} - 排序后的链表头节点
 */
function insertionSortList(head) {
    if (!head || !head.next) return head;
    
    const dummy = new ListNode(0); // 创建虚拟头节点
    let current = head; // 当前待插入的节点
    
    while (current) {
        const next = current.next; // 保存下一个节点
        let prev = dummy; // 从虚拟头节点开始
        
        // 找到合适的插入位置
        while (prev.next && prev.next.val < current.val) {
            prev = prev.next;
        }
        
        // 插入节点
        current.next = prev.next;
        prev.next = current;
        
        current = next; // 处理下一个节点
    }
    
    return dummy.next;
}

希尔排序 ​

希尔排序（Shell Sort）是插入排序的一种改进版本，由 Donald Shell 在 1959 年提出。它通过将数组分成多个子序列，分别对这些子序列进行插入排序，逐步缩小子序列的长度，最终使整个数组基本有序，再进行一次插入排序完成排序。

算法思想 ​

  • 选择间隔序列： 选择一个递减的间隔序列，通常以 n/2 为初始间隔，并逐步减小间隔直到为 1。
  • 间隔排序： 对每个间隔进行插入排序，即对每个子序列进行排序。
  • 逐步缩小间隔： 重复上述过程，不断缩小间隔，直到间隔为 1。
  • 最后一次插入排序： 当间隔为 1 时，进行一次最后的插入排序，完成排序过程。

基础实现 ​

javascript
/**
 * 希尔排序
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function shellSort(arr) {
    const n = arr.length;
    let gap = Math.floor(n / 2); // 初始间隔，取数组长度的一半

    // 当间隔大于 0 时继续排序
    while (gap > 0) {
        // 对每个子序列进行插入排序
        for (let i = gap; i < n; i++) {
            const current = arr[i]; // 当前待插入的元素
            let j = i;
            
            // 在子序列中进行插入排序
            // j >= gap 确保不会越界
            // arr[j - gap] > current 将较大的元素后移
            while (j >= gap && arr[j - gap] > current) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            
            // 将当前元素插入到正确位置
            arr[j] = current;
        }
        
        // 缩小间隔，可以有不同的缩小的策略
        gap = Math.floor(gap / 2); // 每次减半
    }
    return arr;
}

/**
 * 希尔排序 - 使用 Knuth 间隔序列
 * Knuth 序列：1, 4, 13, 40, 121, ... (3^k - 1) / 2
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function shellSortKnuth(arr) {
    const n = arr.length;
    
    // 计算初始间隔
    let gap = 1;
    while (gap < n / 3) {
        gap = gap * 3 + 1; // Knuth 间隔公式
    }
    
    while (gap > 0) {
        for (let i = gap; i < n; i++) {
            const current = arr[i];
            let j = i;
            
            while (j >= gap && arr[j - gap] > current) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            
            arr[j] = current;
        }
        
        gap = Math.floor(gap / 3); // 使用 Knuth 序列缩小
    }
    
    return arr;
}

/**
 * 希尔排序 - 使用 Sedgewick 间隔序列
 * Sedgewick 序列：1, 5, 19, 41, 109, 209, 505, ... (更优的性能)
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function shellSortSedgewick(arr) {
    const n = arr.length;
    
    // 生成 Sedgewick 间隔序列
    const gaps = [];
    let k = 0;
    while (true) {
        if (k % 2 === 0) {
            const gap = 9 * Math.pow(4, k) - 9 * Math.pow(2, k) + 1;
            if (gap > n) break;
            gaps.unshift(gap);
        } else {
            const gap = Math.pow(2, k + 2) * Math.pow(2, k + 2) - 3 * Math.pow(2, k + 2) + 1;
            if (gap > n) break;
            gaps.unshift(gap);
        }
        k++;
    }
    
    // 使用 Sedgewick 间隔序列进行排序
    for (const gap of gaps) {
        for (let i = gap; i < n; i++) {
            const current = arr[i];
            let j = i;
            
            while (j >= gap && arr[j - gap] > current) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            
            arr[j] = current;
        }
    }
    
    return arr;
}

归并排序 ​

归并排序（Merge Sort）是建立在归并操作上的一种高效的排序算法。它采用分治法（Divide and Conquer）的策略，将数组分成两半，分别排序，然后将两个有序数组合并成一个有序数组。

算法思想 ​

  1. 分解：将数组分成两个子数组，递归地对子数组进行排序，直到子数组长度为 1（自然有序）
  2. 合并：将两个有序的子数组合并成一个有序数组

自顶向下归并排序（递归实现） ​

/**
 * 归并排序（自顶向下递归实现）
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function mergeSort(arr) {
    const len = arr.length;
    // 基线条件：数组长度小于等于1时已经有序
    if (len <= 1) {
        return arr;
    }
    
    // 分割数组
    const middle = Math.floor(len / 2);
    const left = arr.slice(0, middle);    // 左半部分
    const right = arr.slice(middle);     // 右半部分
    
    // 递归排序并合并
    return merge(mergeSort(left), mergeSort(right));
}

/**
 * 合并两个有序数组
 * @param {Array} left - 左侧有序数组
 * @param {Array} right - 右侧有序数组
 * @returns {Array} - 合并后的有序数组
 */
function merge(left, right) {
    const result = [];
    let i = 0; // 左数组的指针
    let j = 0; // 右数组的指针
    
    // 比较两个数组的元素，将较小的放入结果数组
    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            result.push(left[i]);
            i++;
        } else {
            result.push(right[j]);
            j++;
        }
    }
    
    // 将剩余的元素全部复制到结果数组
    while (i < left.length) {
        result.push(left[i]);
        i++;
    }
    
    while (j < right.length) {
        result.push(right[j]);
        j++;
    }
    
    return result;
}

/**
 * 归并排序（自底向上迭代实现）
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function mergeSortIterative(arr) {
    const n = arr.length;
    // 使用临时数组辅助合并
    const temp = new Array(n);
    
    // 子数组大小从1开始，每次翻倍
    for (let size = 1; size < n; size *= 2) {
        // 对每一对子数组进行合并
        for (let left = 0; left < n - size; left += 2 * size) {
            const mid = left + size - 1;
            const right = Math.min(left + 2 * size - 1, n - 1);
            
            merge(arr, temp, left, mid, right);
        }
    }
    
    return arr;
}

/**
 * 合并两个有序子数组
 * @param {Array} arr - 原数组
 * @param {Array} temp - 临时数组
 * @param {number} left - 左子数组起始位置
 * @param {number} mid - 左子数组结束位置
 * @param {number} right - 右子数组结束位置
 */
function merge(arr, temp, left, mid, right) {
    // 复制待合并的区间到临时数组
    for (let i = left; i <= right; i++) {
        temp[i] = arr[i];
    }
    
    let i = left;       // 左子数组指针
    let j = mid + 1;    // 右子数组指针
    let k = left;       // 原数组指针
    
    // 合并两个子数组
    while (i <= mid && j <= right) {
        if (temp[i] <= temp[j]) {
            arr[k++] = temp[i++];
        } else {
            arr[k++] = temp[j++];
        }
    }
    
    // 复制剩余元素
    while (i <= mid) {
        arr[k++] = temp[i++];
    }
    
    // 如果右子数组有剩余，不需要复制，因为已经在原数组中
}

/**
 * 链表节点类
 */
class ListNode {
    constructor(val, next = null) {
        this.val = val;
        this.next = next;
    }
}

/**
 * 链表归并排序
 * @param {ListNode} head - 链表头节点
 * @returns {ListNode} - 排序后的链表头节点
 */
function mergeSortList(head) {
    // 基线条件：空链表或只有一个节点
    if (!head || !head.next) {
        return head;
    }
    
    // 使用快慢指针找到链表中间节点
    let slow = head;
    let fast = head.next;
    
    while (fast && fast.next) {
        slow = slow.next;
        fast = fast.next.next;
    }
    
    // 分割链表
    const mid = slow.next;
    slow.next = null;
    
    // 递归排序
    const left = mergeSortList(head);
    const right = mergeSortList(mid);
    
    // 合并两个有序链表
    return mergeLists(left, right);
}

/**
 * 合并两个有序链表
 * @param {ListNode} l1 - 第一个有序链表
 * @param {ListNode} l2 - 第二个有序链表
 * @returns {ListNode} - 合并后的有序链表
 */
function mergeLists(l1, l2) {
    const dummy = new ListNode(0);
    let current = dummy;
    
    while (l1 && l2) {
        if (l1.val <= l2.val) {
            current.next = l1;
            l1 = l1.next;
        } else {
            current.next = l2;
            l2 = l2.next;
        }
        current = current.next;
    }
    
    // 连接剩余部分
    current.next = l1 || l2;
    
    return dummy.next;
}

快速排序 ​

快速排序（Quick Sort）是由东尼·霍尔（Tony Hoare）在 1960 年提出的一种高效的排序算法。它采用分治法（Divide and Conquer）的思想，通过选择一个基准元素，将数组分为两部分，然后递归地对两部分进行排序。

算法思想 ​

  1. 选择基准（Pivot）：从数组中选择一个元素作为基准
  2. 分区（Partition）：将数组中小于基准元素的元素放到基准左边，大于基准的放到右边
  3. 递归排序：对左右两个子数组分别递归地应用快速排序

基础实现 ​

javascript
/**
 * 快速排序基础实现
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function quickSort(arr) {
    // 基线条件：数组长度小于等于1时已经有序
    if (arr.length <= 1) {
        return arr;
    }
    
    // 选择基准元素（这里选择第一个元素）
    const pivot = arr[0];
    const left = [];   // 存放比基准小的元素
    const right = [];  // 存放比基准大的元素
    const equal = [];  // 存放和基准相等的元素（确保稳定性）
    
    // 分区操作
    for (let i = 0; i < arr.length; i++) {
        if (arr[i] < pivot) {
            left.push(arr[i]);
        } else if (arr[i] > pivot) {
            right.push(arr[i]);
        } else {
            equal.push(arr[i]);
        }
    }
    
    // 递归排序左右两部分，并与基准合并
    return [...quickSort(left), ...equal, ...quickSort(right)];
}

/**
 * 原地快速排序（Lomuto 分区方案）
 * @param {Array} arr - 待排序数组
 * @param {number} low - 起始索引
 * @param {number} high - 结束索引
 * @returns {Array} - 排序后的数组
 */
function quickSortInPlace(arr, low = 0, high = arr.length - 1) {
    if (low < high) {
        // 分区操作，返回基准元素的索引
        const pivotIndex = partition(arr, low, high);
        
        // 递归排序左半部分
        quickSortInPlace(arr, low, pivotIndex - 1);
        // 递归排序右半部分
        quickSortInPlace(arr, pivotIndex + 1, high);
    }
    
    return arr;
}

/**
 * Lomuto 分区方案
 * @param {Array} arr - 数组
 * @param {number} low - 起始索引
 * @param {number} high - 结束索引
 * @returns {number} - 基准元素的最终位置
 */
function partition(arr, low, high) {
    // 选择最后一个元素作为基准
    const pivot = arr[high];
    let i = low - 1; // i 是小于基准的区域的边界
    
    // 遍历并重新排列元素
    for (let j = low; j < high; j++) {
        // 如果当前元素小于或等于基准
        if (arr[j] <= pivot) {
            i++;
            // 交换元素
            [arr[i], arr[j]] = [arr[j], arr[i]];
        }
    }
    
    // 将基准放到正确的位置
    [arr[i + 1], arr[high]] = [arr[high], arr[i + 1]];
    return i + 1;
}

/**
 * 快速排序（Hoare 分区方案）
 * Hoare 分区通常比 Lomuto 分区更高效
 * @param {Array} arr - 待排序数组
 * @param {number} low - 起始索引
 * @param {number} high - 结束索引
 * @returns {Array} - 排序后的数组
 */
function quickSortHoare(arr, low = 0, high = arr.length - 1) {
    if (low < high) {
        // Hoare 分区
        const pivotIndex = hoarePartition(arr, low, high);
        
        // Hoare 分区返回的索引与 Lomuto 不同
        quickSortHoare(arr, low, pivotIndex);
        quickSortHoare(arr, pivotIndex + 1, high);
    }
    
    return arr;
}

/**
 * Hoare 分区函数
 * @param {Array} arr - 数组
 * @param {number} low - 起始索引
 * @param {number} high - 结束索引
 * @returns {number} - 分区索引
 */
function hoarePartition(arr, low, high) {
    // 选择第一个元素作为基准
    const pivot = arr[low];
    let i = low - 1;
    let j = high + 1;
    
    while (true) {
        // 找到左边大于等于 pivot 的元素
        do {
            i++;
        } while (arr[i] < pivot);
        
        // 找到右边小于等于 pivot 的元素
        do {
            j--;
        } while (arr[j] > pivot);
        
        // 如果指针交叉，返回 j 作为分区点
        if (i >= j) return j;
        
        // 交换元素
        [arr[i], arr[j]] = [arr[j], arr[i]];
    }
}

/**
 * 快速排序 - 三数取中法优化
 * @param {Array} arr - 待排序数组
 * @param {number} low - 起始索引
 * @param {number} high - 结束索引
 * @returns {Array} - 排序后的数组

function quickSortOptimized(arr, low = 0, high = arr.length - 1) {
    if (low < high) {
        // 三数取中法选择基准
        const pivotIndex = medianOfThree(arr, low, high);
        
        // 将基准交换到末尾
        [arr[pivotIndex], arr[high]] = [arr[high], arr[pivotIndex]];
        
        // 分区
        const partitionIndex = partition(arr, low, high);
        
        // 递归排序
        quickSortOptimized(arr, low, partitionIndex - 1);
        quickSortOptimized(arr, partitionIndex + 1, high);
    }
    
    return arr;
}

/**
 * 三数取中法：选择左端、中间、右端三个元素的中值作为基准
 * @param {Array} arr - 数组
 * @param {number} low - 起始索引
 * @param {number} high - 结束索引
 * @returns {number} - 基准元素的索引
 */
function medianOfThree(arr, low, high) {
    const mid = Math.floor((low + high) / 2);
    
    // 对三个元素进行排序
    if (arr[low] > arr[mid]) {
        [arr[low], arr[mid]] = [arr[mid], arr[low]];
    }
    if (arr[low] > arr[high]) {
        [arr[low], arr[high]] = [arr[high], arr[low]];
    }
    if (arr[mid] > arr[high]) {
        [arr[mid], arr[high]] = [arr[high], arr[mid]];
    }
    
    // 返回中间值的索引
    return mid;
}

堆排序 ​

堆排序（Heap Sort）是一种基于二叉堆数据结构的排序算法。它利用了堆这种数据结构的特性：堆顶元素总是最大（或最小）的元素。堆排序不需要额外的存储空间，是原地排序算法。

算法思想 ​

  1. 建堆：将无序序列构建成一个大顶堆（或小顶堆）
  2. 排序：不断将堆顶元素（最大值）与堆末尾元素交换，然后调整剩余元素成为一个新的堆

基础实现 ​

javascript
/**
 * 堆排序
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function heapSort(arr) {
    const n = arr.length;
    
    // 构建大顶堆（从最后一个非叶子节点开始）
    for (let i = Math.floor(n / 2) - 1; i >= 0; i--) {
        heapify(arr, n, i);
    }
    
    // 逐个提取堆顶元素
    for (let i = n - 1; i > 0; i--) {
        // 将堆顶元素（最大值）移动到数组末尾
        [arr[0], arr[i]] = [arr[i], arr[0]];
        
        // 调整剩余元素成为新的堆
        heapify(arr, i, 0);
    }
    
    return arr;
}

/**
 * 堆调整函数
 * @param {Array} arr - 数组
 * @param {number} n - 堆的大小
 * @param {number} i - 当前调整的节点索引
 */
function heapify(arr, n, i) {
    let largest = i;        // 初始化最大值为当前节点
    const left = 2 * i + 1;  // 左子节点索引
    const right = 2 * i + 2; // 右子节点索引
    
    // 如果左子节点大于当前最大值
    if (left < n && arr[left] > arr[largest]) {
        largest = left;
    }
    
    // 如果右子节点大于当前最大值
    if (right < n && arr[right] > arr[largest]) {
        largest = right;
    }
    
    // 如果最大值不是当前节点，交换并继续调整
    if (largest !== i) {
        [arr[i], arr[largest]] = [arr[largest], arr[i]];
        
        // 递归调整受影响的子树
        heapify(arr, n, largest);
    }
}

/**
 * 基于堆的优先队列类
 */
class MaxPriorityQueue {
    constructor() {
        this.heap = [];
    }
    
    // 获取父节点索引
    getParentIndex(index) {
        return Math.floor((index - 1) / 2);
    }
    
    // 获取左子节点索引
    getLeftChildIndex(index) {
        return 2 * index + 1;
    }
    
    // 获取右子节点索引
    getRightChildIndex(index) {
        return 2 * index + 2;
    }
    
    // 插入元素
    insert(value) {
        this.heap.push(value);
        this.bubbleUp(this.heap.length - 1);
    }
    
    // 上浮操作
    bubbleUp(index) {
        while (index > 0) {
            const parentIndex = this.getParentIndex(index);
            
            if (this.heap[parentIndex] < this.heap[index]) {
                [this.heap[parentIndex], this.heap[index]] = 
                    [this.heap[index], this.heap[parentIndex]];
                index = parentIndex;
            } else {
                break;
            }
        }
    }
    
    // 取出最大值
    extractMax() {
        if (this.heap.length === 0) {
            return null;
        }
        
        if (this.heap.length === 1) {
            return this.heap.pop();
        }
        
        const max = this.heap[0];
        this.heap[0] = this.heap.pop();
        this.bubbleDown(0);
        
        return max;
    }
    
    // 下沉操作
    bubbleDown(index) {
        while (true) {
            const leftChild = this.getLeftChildIndex(index);
            const rightChild = this.getRightChildIndex(index);
            let largest = index;
            
            if (leftChild < this.heap.length && 
                this.heap[leftChild] > this.heap[largest]) {
                largest = leftChild;
            }
            
            if (rightChild < this.heap.length && 
                this.heap[rightChild] > this.heap[largest]) {
                largest = rightChild;
            }
            
            if (largest !== index) {
                [this.heap[index], this.heap[largest]] = 
                    [this.heap[largest], this.heap[index]];
                index = largest;
            } else {
                break;
            }
        }
    }
    
    // 查看最大值
    peek() {
        return this.heap.length > 0 ? this.heap[0] : null;
    }
    
    // 获取大小
    size() {
        return this.heap.length;
    }
}

// 使用优先队列进行排序
function heapSortWithPriorityQueue(arr) {
    const pq = new MaxPriorityQueue();
    
    // 将所有元素插入优先队列
    for (const item of arr) {
        pq.insert(item);
    }
    
    const result = [];
    // 按顺序取出所有元素
    while (pq.size() > 0) {
        result.push(pq.extractMax());
    }
    
    return result;
}

计数排序 ​

计数排序（Counting Sort）是一种非比较性的排序算法，它的核心思想是统计每个元素出现的次数，然后根据统计结果将元素放回正确的位置。计数排序的时间复杂度为 O(n + k)，其中 k 是数据的范围。

算法思想 ​

  1. 统计频率：统计每个元素出现的次数
  2. 计算前缀和：计算每个元素的累计出现次数，确定元素的位置
  3. 放置元素：根据统计结果将元素放回原数组

基础实现 ​

javascript
/**
 * 计数排序
 * @param {Array} arr - 待排序数组
 * @returns {Array} - 排序后的数组
 */
function countingSort(arr) {
    if (arr.length === 0) return arr;
    
    // 1. 找出数组中的最大值和最小值
    const max = Math.max(...arr);
    const min = Math.min(...arr);
    
    // 2. 创建计数数组，大小为 max - min + 1
    const count = new Array(max - min + 1).fill(0);
    
    // 3. 统计每个元素出现的次数
    for (const num of arr) {
        count[num - min]++;
    }
    
    // 4. 计算前缀和（确定每个元素的最终位置）
    for (let i = 1; i < count.length; i++) {
        count[i] += count[i - 1];
    }
    
    // 5. 从后向前遍历原数组，将元素放到正确的位置
    const result = new Array(arr.length);
    for (let i = arr.length - 1; i >= 0; i--) {
        const num = arr[i];
        const pos = count[num - min] - 1;
        result[pos] = num;
        count[num - min]--;
    }
    
    return result;
}

桶排序 ​

桶排序（Bucket Sort）是一种分布排序算法，它将数据分到不同的桶中，然后对每个桶分别进行排序，最后将所有桶合并得到有序序列。桶排序适用于数据分布均匀的情况。

算法思想 ​

  1. 创建桶：根据数据范围和分布创建 n 个空桶
  2. 分配元素：将每个元素放入对应的桶中
  3. 排序桶：对每个桶进行排序（可以使用任何排序算法）
  4. 合并桶：按顺序将所有桶中的元素合并

基础实现 ​

javascript
/**
 * 桶排序
 * @param {Array} arr - 待排序数组
 * @param {number} bucketSize - 每个桶的大小
 * @returns {Array} - 排序后的数组
 */
function bucketSort(arr, bucketSize = 5) {
    if (arr.length === 0) return arr;
    
    // 1. 找出数组中的最大值和最小值
    const max = Math.max(...arr);
    const min = Math.min(...arr);
    
    // 2. 计算桶的数量
    const bucketCount = Math.floor((max - min) / bucketSize) + 1;
    const buckets = Array.from({ length: bucketCount }, () => []);
    
    // 3. 将元素分配到对应的桶中
    for (const num of arr) {
        const bucketIndex = Math.floor((num - min) / bucketSize);
        buckets[bucketIndex].push(num);
    }
    
    // 4. 对每个桶进行排序
    for (let i = 0; i < buckets.length; i++) {
        if (buckets[i].length > 0) {
            // 使用插入排序对小桶进行排序
            insertionSort(buckets[i]);
        }
    }
    
    // 5. 合并所有桶
    const result = [];
    for (const bucket of buckets) {
        result.push(...bucket);
    }
    
    return result;
}

// 辅助函数：插入排序
function insertionSort(arr) {
    for (let i = 1; i < arr.length; i++) {
        let current = arr[i];
        let j = i - 1;
        
        while (j >= 0 && arr[j] > current) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = current;
    }
    return arr;
}

/**
 * 使用哈希表实现的桶排序
 * @param {Array} arr - 待排序数组
 * @param {Function} hashFunc - 哈希函数
 * @returns {Array} - 排序后的数组
 */
function bucketSortWithHash(arr, hashFunc) {
    if (arr.length === 0) return arr;
    
    // 创建桶
    const buckets = new Map();
    
    // 将元素分配到桶中
    for (const item of arr) {
        const key = hashFunc(item);
        
        if (!buckets.has(key)) {
            buckets.set(key, []);
        }
        buckets.get(key).push(item);
    }
    
    // 对每个桶进行排序
    const sortedBuckets = [];
    for (const [key, bucket] of buckets) {
        bucket.sort((a, b) => a - b);
        sortedBuckets.push(...bucket);
    }
    
    return sortedBuckets;
}

    for (const num of arr) {
        // 确保数字在 0-1 范围内
        if (num < 0 || num > 1) {
            throw new Error('Float bucket sort requires numbers between 0 and 1');
        }
        
        const bucketIndex = Math.floor(num * bucketCount);
        buckets[bucketIndex].push(num);
    }
    
    // 对每个桶进行排序
    for (let i = 0; i < buckets.length; i++) {
        buckets[i].sort((a, b) => a - b);
    }
    
    // 合并桶
    const result = [];
    for (const bucket of buckets) {
        result.push(...bucket);
    }
    
    return result;
}

const floatArray = [0.12, 0.45, 0.23, 0.67, 0.34, 0.89, 0.56, 0.78];
console.log(bucketSortFloat([...floatArray]));
// 输出 [0.12, 0.23, 0.34, 0.45, 0.56, 0.67, 0.78, 0.89]

基数排序 ​

基数排序（Radix Sort）是一种非比较性的排序算法，它按照数字的位数进行排序。基数排序可以处理整数、字符串等数据，适用于位数固定的数据排序。

算法思想 ​

  1. 按位排序：从最低位（个位）开始，到最高位依次进行排序
  2. 稳定排序：每次排序都要保持相等元素的相对位置
  3. 递进排序：依次对每一位进行排序，最终得到有序序列

基础实现（LSD - 最显著位优先） ​

javascript
/**
 * 基数排序（LSD - Least Significant Digit）
 * @param {Array} arr - 待排序数组
 * @param {number} maxDigits - 最大位数
 * @returns {Array} - 排序后的数组
 */
function radixSort(arr, maxDigits = 10) {
    if (arr.length === 0) return arr;
    
    // 找出最大值，确定最大位数
    const max = Math.max(...arr);
    const maxDigitsActual = Math.floor(Math.log10(max)) + 1;
    
    // 从个位开始，每一位进行计数排序
    for (let exp = 1; exp <= Math.pow(10, maxDigitsActual - 1); exp *= 10) {
        countingSortByDigit(arr, exp);
    }
    
    return arr;
}

/**
 * 按特定位数进行计数排序
 * @param {Array} arr - 数组
 * @param {number} exp - 位权（1, 10, 100, ...）
 */
function countingSortByDigit(arr, exp) {
    const n = arr.length;
    const output = new Array(n);
    const count = new Array(10).fill(0);
    
    // 统计每个数字出现的次数
    for (let i = 0; i < n; i++) {
        const digit = Math.floor(arr[i] / exp) % 10;
        count[digit]++;
    }
    
    // 计算前缀和
    for (let i = 1; i < 10; i++) {
        count[i] += count[i - 1];
    }
    
    // 从后向前放置元素，保持稳定性
    for (let i = n - 1; i >= 0; i--) {
        const digit = Math.floor(arr[i] / exp) % 10;
        output[count[digit] - 1] = arr[i];
        count[digit]--;
    }
    
    // 将排序结果复制回原数组
    for (let i = 0; i < n; i++) {
        arr[i] = output[i];
    }
}

console.log(radixSort([...array])); // 输出 [2, 24, 45, 66, 75, 90, 170, 802]

字符串基数排序 ​

javascript
/**
 * 字符串基数排序
 * @param {Array} arr - 待排序的字符串数组
 * @param {number} maxLen - 最大字符串长度
 * @returns {Array} - 排序后的字符串数组
 */
function radixSortStrings(arr, maxLen = 0) {
    if (arr.length === 0) return arr;
    
    // 找出最大长度
    maxLen = Math.max(maxLen, ...arr.map(str => str.length));
    
    // 从最后一位开始向前排序
    for (let pos = maxLen - 1; pos >= 0; pos--) {
        countingSortByChar(arr, pos);
    }
    
    return arr;
}

/**
 * 按字符位置进行计数排序
 * @param {Array} arr - 字符串数组
 * @param {number} pos - 字符位置
 */
function countingSortByChar(arr, pos) {
    const n = arr.length;
    const output = new Array(n);
    const count = new Array(256).fill(0); // ASCII 范围
    
    // 统计每个字符出现的次数
    for (let i = 0; i < n; i++) {
        const char = pos < arr[i].length ? arr[i].charCodeAt(pos) : 0; // 空字符用0表示
        count[char]++;
    }
    
    // 计算前缀和
    for (let i = 1; i < 256; i++) {
        count[i] += count[i - 1];
    }
    
    // 从后向前放置元素
    for (let i = n - 1; i >= 0; i--) {
        const char = pos < arr[i].length ? arr[i].charCodeAt(pos) : 0;
        output[count[char] - 1] = arr[i];
        count[char]--;
    }
    
    // 复制回原数组
    for (let i = 0; i < n; i++) {
        arr[i] = output[i];
    }
}

MSD（最显著位优先）基数排序 ​

javascript
/**
 * MSD 基数排序（Most Significant Digit）
 * @param {Array} arr - 待排序数组
 * @param {number} low - 起始索引
 * @param {number} high - 结束索引
 * @param {number} exp - 位权
 * @returns {Array} - 排序后的数组
 */
function radixSortMSD(arr, low = 0, high = arr.length - 1, exp = 1) {
    if (low >= high) return arr;
    
    const count = new Array(10).fill(0);
    const output = new Array(high - low + 1);
    
    // 统计频率
    for (let i = low; i <= high; i++) {
        const digit = Math.floor(arr[i] / exp) % 10;
        count[digit]++;
    }
    
    // 计算前缀和
    for (let i = 1; i < 10; i++) {
        count[i] += count[i - 1];
    }
    
    // 从后向前放置
    for (let i = high; i >= low; i--) {
        const digit = Math.floor(arr[i] / exp) % 10;
        output[count[digit] - 1] = arr[i];
        count[digit]--;
    }
    
    // 复制回原数组
    for (let i = 0; i < output.length; i++) {
        arr[low + i] = output[i];
    }
    
    // 递归排序每个桶
    let prev = low;
    for (let i = 0; i < 10; i++) {
        if (count[i] > prev) {
            radixSortMSD(arr, prev, prev + count[i] - 1, exp * 10);
            prev += count[i];
        }
    }
    
    return arr;
}

性能分析 ​

  • 时间复杂度：

    • 最佳情况：O(d(n + k)) - d 是数字位数
    • 平均情况：O(d(n + k))
    • 最坏情况：O(d(n + k))

  • 稳定性：稳定 - 相等元素的相对位置不会改变

  • 适用场景：

    • 整数排序（特别是位数固定的情况）
    • 字符串排序
    • 邮编、电话号码等固定长度数据的排序
    • 需要稳定排序的场景

  • 关键因素：

    • 位数 d 的大小：位数越多，时间复杂度越高
    • 数据分布：对位数相同的数据效果最好
    • 基数的选择：通常使用 10（十进制）

  • 与其他排序算法的比较：

    • 相比比较排序算法，基数排序的时间复杂度与数据规模成线性关系
    • 适合固定长度的数据，如身份证号、手机号等
    • 不适合大数据范围或位数不确定的情况

总结 ​

排序算法是计算机科学中的基础内容，每种算法都有其特点和适用场景。选择合适的排序算法需要考虑以下几个因素：

选择排序算法的考虑因素 ​

  1. 数据规模：

    • 小规模数据（<100）：插入排序、选择排序
    • 中等规模数据（100-10000）：快速排序、归并排序、堆排序
    • 大规模数据（>10000）：快速排序、归并排序

  2. 数据特点：

    • 基本有序：插入排序（O(n)）
    • 随机分布：快速排序（平均 O(n log n)）
    • 几乎有序：插入排序、希尔排序
    • 范围较小：计数排序、桶排序
    • 固定长度：基数排序

  3. 内存限制：

    • 内存有限：堆排序、原地快速排序
    • 内存充足：归并排序、基数排序

  4. 稳定性要求：

    • 需要稳定：归并排序、计数排序、桶排序、插入排序
    • 不需要稳定：快速排序、堆排序、选择排序

实际应用中的选择 ​

  • JavaScript 内置的 sort()：使用 V8 引擎的 TimSort（归并排序 + 插入排序的混合算法）
  • 大多数编程语言：对小数组使用插入排序，大数组使用快速排序
  • 数据库系统：使用 B+ 树等数据结构进行排序
  • 大数据处理：使用外部排序、MapReduce 等分布式排序算法

排序算法的发展历程 ​

  1. 早期算法（1950s）：冒泡排序、选择排序、插入排序
  2. 分治算法（1960s）：快速排序、归并排序、堆排序
  3. 线性时间算法（1970s）：计数排序、桶排序、基数排序
  4. 现代优化（1990s）：TimSort、Introsort 等

未来发展趋势 ​

  1. 并行化：利用多核 CPU、GPU 进行并行排序
  2. 分布式排序：适应大数据时代的分布式排序算法
  3. 机器学习辅助：使用机器学习预测最佳排序算法
  4. 量子排序：量子计算环境下的排序算法

排序算法的学习建议 ​

  1. 从简单开始：先掌握插入排序、选择排序等基础算法
  2. 理解核心思想：重点理解分治、分桶、计数等核心思想
  3. 动手实现：亲手实现每种算法，加深理解
  4. 分析性能：分析不同算法在不同数据下的表现
  5. 实际应用：在项目中尝试使用不同的排序算法
