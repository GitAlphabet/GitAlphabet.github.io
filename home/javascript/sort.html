# 十种经典排序

# 冒泡排序

冒泡排序（Bubble Sort）是一种简单直观的排序算法。冒泡排序算法的步骤描述如下：

  • 比较相邻的元素。如果第一个比第二个大，就交换他们两个。
  • 对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对。这步做完后，最后的元素会是最大的数。
  • 针对所有的元素重复以上的步骤，除了最后一个。
  • 持续每次对越来越少的元素重复上面的步骤，直到没有任何一对数字需要比较。
function bubbleSort(arr) {
    let len = arr.length;
    for (let i = 0; i < len - 1; i++) {
        for (let j = 0; j < len - 1 - i; j++) {
            if (arr[j] > arr[j+1]) {        // 相邻元素两两对比
                let temp = arr[j+1];        // 元素交换
                arr[j+1] = arr[j];
                arr[j] = temp;
            }
        }
    }
    return arr;
}

# 选择排序

算法思想

  • 初始状态： 将序列中的第一个元素视为最小值。
  • 遍历未排序部分： 遍历剩余的未排序部分，找到其中最小的元素。
  • 交换操作： 将最小元素与未排序部分的第一个元素交换位置，即将其放到已排序部分的末尾。
  • 重复操作： 重复以上步骤，直到整个序列排序完成。
function selectionSort(arr) {
    const n = arr.length;
    for (let i = 0; i < n - 1; i++) {
        let minIndex = i;
        for (let j = i + 1; j < n; j++) {
            if (arr[j] < arr[minIndex]) {
                minIndex = j; // 找到未排序部分的最小元素下标
            }
        }
        // 将未排序部分的最小元素与已排序部分的末尾交换位置
        [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]];
    }
    return arr;
}

// 示例
const array = [5, 3, 7, 2, 9, 1, 6];
const sortedArray = selectionSort(array);
console.log(sortedArray); // 输出 [1, 2, 3, 5, 6, 7, 9]

解释 在每一轮遍历中，选择未排序部分的最小元素，并将其与已排序部分的末尾交换位置。 通过不断地选择最小元素，并逐步将其放到已排序部分的末尾，最终完成整个序列的排序。

性能 选择排序的时间复杂度始终为O(n^2)，其中n是待排序序列的长度，无论输入数据的分布情况如何。尽管选择排序在实践中很少被用作排序算法，但它仍然有其简单直观的优点。

# 插入排序

算法思想

  • 初始状态： 将序列中的第一个元素视为已排序部分。
  • 遍历未排序部分： 从未排序部分选择一个元素，将其插入到已排序部分的合适位置，使得已排序部分仍然有序。
  • 插入操作： 将选定的元素与已排序部分的元素逐个比较，找到合适的插入位置，并将其插入。
  • 重复操作： 重复以上步骤，直到整个序列排序完成。
function insertionSort(arr) {
    const n = arr.length;
    for (let i = 1; i < n; i++) {
        let current = arr[i]; // 当前待插入的元素
        let j = i - 1; // 已排序部分的最后一个元素的下标
        while (j >= 0 && arr[j] > current) {
            arr[j + 1] = arr[j]; // 将大于当前元素的元素后移
            j--;
        }
        arr[j + 1] = current; // 将当前元素插入到合适位置
    }
    return arr;
}

// 示例
const array = [5, 3, 7, 2, 9, 1, 6];
const sortedArray = insertionSort(array);
console.log(sortedArray); // 输出 [1, 2, 3, 5, 6, 7, 9]

解释

  • 在每一轮遍历中，从未排序部分选择一个元素，然后将其插入到已排序部分的合适位置。
  • 插入操作的过程是将当前元素与已排序部分的元素逐个比较，并将大于当前元素的元素后移，直到找到合适的插入位置。
  • 通过不断地选择并插入元素，最终完成整个序列的排序。

性能 插入排序的时间复杂度最好情况为O(n)，即当输入序列已经有序时。最坏情况下的时间复杂度为O(n^2)，其中n是待排序序列的长度。尽管插入排序在大规模数据集上性能较差，但在小规模数据集和基本有序的数据集上表现良好。

# 希尔排序

希尔排序（Shell Sort）是插入排序的一种改进版本，也被称为缩小增量排序。它的核心思想是将待排序的数组按照一定间隔分成若干个子序列，对每个子序列进行插入排序，然后逐步缩小间隔，直到间隔为1时进行最后一次插入排序，从而实现对整个数组的排序。

算法思想

  • 选择间隔序列： 选择一个递减的间隔序列，通常以n/2为初始间隔，并逐步减小间隔直到为1。
  • 间隔排序： 对每个间隔进行插入排序，即对每个子序列进行排序。
  • 逐步缩小间隔： 重复上述过程，不断缩小间隔，直到间隔为1。
  • 最后一次插入排序： 当间隔为1时，进行一次最后的插入排序，完成排序过程。
function shellSort(arr) {
    const n = arr.length;
    let gap = Math.floor(n / 2); // 初始间隔

    while (gap > 0) {
        for (let i = gap; i < n; i++) {
            const current = arr[i];
            let j = i;
            while (j >= gap && arr[j - gap] > current) {
                arr[j] = arr[j - gap];
                j -= gap;
            }
            arr[j] = current;
        }
        gap = Math.floor(gap / 2); // 缩小间隔
    }
    return arr;
}

// 示例
const array = [5, 3, 7, 2, 9, 1, 6];
const sortedArray = shellSort(array);
console.log(sortedArray); // 输出 [1, 2, 3, 5, 6, 7, 9]

# 归并排序

归并排序（Merge sort）是建立在归并操作上的一种有效的排序算法。归并排序算法的步骤描述如下：

  • 申请空间，使其大小为两个已经排序序列之和，该空间用来存放合并后的序列；
  • 设定两个指针，最初位置分别为两个已经排序序列的起始位置；
  • 比较两个指针所指向的元素，选择相对小的元素放入到合并空间，并移动指针到下一位置；
  • 重复步骤 3 直到某一指针达到序列尾；
  • 将另一序列剩下的所有元素直接复制到合并序列尾。
function mergeSort(arr) {  // 采用自上而下的递归方法
    var len = arr.length;
    if(len < 2) {
        return arr;
    }
    var middle = Math.floor(len / 2),
        left = arr.slice(0, middle),
        right = arr.slice(middle);
    return merge(mergeSort(left), mergeSort(right));
}
 
function merge(left, right){
    var result = [];
    while (left.length && right.length) {
        if (left[0] <= right[0]) {
            result.push(left.shift());
        } else {
            result.push(right.shift());
        }
    }
    while (left.length)
        result.push(left.shift());
 
    while (right.length)
        result.push(right.shift());
    return result;
}

# 快速排序

快速排序是由东尼·霍尔所发展的一种排序算法。它基于分治法（Divide and Conquer）的思想。

算法思想

  • 选择基准元素（pivot）： 从数组中选择一个元素作为基准（通常选择第一个或最后一个元素）。
  • 划分操作： 将数组中小于基准元素的放到基准元素的左边，大于基准元素的放到右边，基准元素放到中间，称之为“划分操作”。
  • 递归调用： 对划分后的两个子数组分别递归地应用快速排序算法。
function quickSort(arr) {
    if (arr.length <= 1) {
        return arr; // 基线条件：数组为空或只包含一个元素，直接返回
    } else {
        const pivot = arr[0]; // 选择第一个元素作为基准
        const left = [];
        const right = [];
        
        for (let i = 1; i < arr.length; i++) {
            if (arr[i] < pivot) {
                left.push(arr[i]); // 比基准小的放到左边
            } else {
                right.push(arr[i]); // 比基准大的放到右边
            }
        }
        return quickSort(left).concat(pivot, quickSort(right)); // 递归调用，并拼接结果
    }
}
// 示例
const array = [5, 3, 7, 2, 9, 1, 6];
const sortedArray = quickSort(array);
console.log(sortedArray); // 输出 [1, 2, 3, 5, 6, 7, 9]

解释

  • 此实现中，我们选择数组的第一个元素作为基准（pivot）。
  • 遍历数组除了基准之外的所有元素，将比基准小的元素放到左边，比基准大的放到右边。
  • 然后，对左右两个子数组分别递归地应用快速排序，并将结果合并起来。

性能 快速排序的平均时间复杂度为O(n log n)，在最坏情况下为O(n^2)（当选择的基准不好时）。然而，在实践中，它通常比其他O(n log n)算法更快，因为它的内部循环可以在大部分计算机体系结构上很有效地被实现。
